Селекторы

Виды селекторов:

1. Селектор по тегу (элементу) - Этот селектор CSS применяет стили ко всем элементам с одинаковым тегом.

2. Селектор по идентификатору (id) - Селектор по идентификатору обозначается символом # (решётка) и применяет стили к элементу, для которого задан атрибут id с соответствующим значением.

3. Селектор по классу (class) - CSS-селектор по классу выбирает элементы, для которых назначен атрибут class с соответствующим значением.

4. За несколькими классами - CSS-селекторы можно сгруппировать, чтобы применить стили к нескольким группам и/или классам элементов.

5. Селекторы по вложенности

6. Универсальный селектор - Он применяет стили ко всем элементам страницы и обозначается символом * (звёздочка).
